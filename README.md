Teensy 3.0

Installation för teensy krävs. Instruktion:
https://www.pjrc.com/teensy/td_download.html

**Kablar**
Siffrorna syftar till raden på breadboarden där kabeln sitter
Färgerna syftar på kabelfärgerna

**Skärm**
GND -  orange
VCC +  gul
SCL 23 grön
SDA 22 blå

**Joystick**
GND -  svart
+5V +  röd
VRx 19 gul
VRy 18 brun
